# AWS Project 05 - AWS CLI Administration Lab

## Objective
Learn how to manage AWS services using the AWS Command Line Interface (CLI).

## Services Used
- IAM
- AWS CLI
- EC2
- S3
- STS

## Tasks Performed
1. Installed AWS CLI
2. Created IAM User
3. Generated Access Keys
4. Configured AWS CLI
5. Verified authentication using STS
6. Listed EC2 instances
7. Listed AWS regions
8. Listed S3 buckets
9. Verified CLI configuration

## Outcome
Successfully authenticated AWS CLI and executed AWS management commands from the terminal.
